## 🐛 已修复的关键问题

### 1. optimize_strategy 调用参数错误 ✅

**问题**：调用 `optimize_strategy(strategy_id=..., objective=...)` 但函数签名不匹配

**修复**：
```python
# 获取策略规格
strategy_data = get_strategy(strategy_id)
spec = StrategySpec(**strategy_data['spec'])
config = OptimizationConfig(objective="sharpe", search_method="grid", max_trials=30)

# 正确调用
optimization_result = loop.run_until_complete(optimize_strategy(spec, config))
```

### 2. 异步函数在同步代码中调用 ✅

**问题**：`optimize_strategy` 是 async 函数，在同步方法中直接调用会失败

**修复**：
```python
# 获取或创建事件循环
try:
    loop = asyncio.get_event_loop()
except RuntimeError:
    loop = asyncio.new_event_loop()
    asyncio.set_event_loop(loop)

# 在同步代码中执行异步函数
optimization_result = loop.run_until_complete(optimize_strategy(spec, config))
```

### 3. candidate_spec 空字典处理 ✅

**问题**：rejected 时 `candidate_spec` 为空字典 `{}`，需转为 `None`

**修复**：
```python
# 使用 truthiness 判断
candidate_spec = optimization_result.candidate_spec if optimization_result.candidate_spec else None

# 结果：{} -> None, {...} -> {...}
```
