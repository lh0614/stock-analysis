## 🔍 发现的问题

### 问题 1: optimize_strategy 调用不匹配 ⚠️ 已修复

**问题描述：**
- `strategy_cycle.py` 中调用 `optimize_strategy(strategy_id=..., objective=...)`
- 但实际函数签名是 `async def optimize_strategy(strategy_spec: StrategySpec, config: OptimizationConfig, ...)`

**影响：**
- 优化触发会直接失败
- 衰减策略无法自动生成优化候选

**修复方案：**
```python
# 修复前（错误）
optimization_result = optimize_strategy(
    strategy_id=strategy_id,
    objective="improve_health"
)

# 修复后（正确）
strategy_data = get_strategy(strategy_id)
spec = StrategySpec(**strategy_data['spec'])
config = OptimizationConfig(
    objective="sharpe",
    search_method="grid",
    max_trials=30
)
optimization_result = loop.run_until_complete(
    optimize_strategy(spec, config)
)
```

**修复状态：** ✅ 已完成

---

### 问题 2: 异步函数在同步代码中调用 ⚠️ 已修复

**问题描述：**
- `optimize_strategy` 是 async 函数
- 在 `strategy_cycle.py` 的同步方法 `_execute_cycle()` 中调用
- 缺少 await 或事件循环处理

**影响：**
- 优化调用会抛出异常
- 返回的是协程对象而非结果

**修复方案：**
使用 `asyncio.get_event_loop().run_until_complete()` 在同步代码中执行异步函数

**修复状态：** ✅ 已完成
