## 🚀 使用方法

### 快速启动
```bash
./start.sh
```

### 访问系统
- 智能选股：http://localhost:5173/intelligent-screener
- 策略库：http://localhost:5173/strategy-library
- API文档：http://localhost:8000/docs

### 创建策略
1. 输入策略名称
2. 选择投资周期
3. 添加筛选条件（25个因子可选）
4. 添加排序规则
5. 点击"选股并回测"
6. 查看评级和回测结果
7. 优质策略自动保存到策略库
