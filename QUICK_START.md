# 快速开始指南

**本地智能股票分析平台 - 5 分钟上手**

---

## 🚀 启动服务

### 1. 启动后端（必须先启动）

```bash
cd backend
python -m uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

**验证启动成功**：
```bash
curl http://localhost:8000/health
# 期望返回: {"status":"healthy","timestamp":"..."}
```

**API 文档**：打开浏览器访问 http://localhost:8000/docs

---

### 2. 启动前端

```bash
cd frontend
npm install  # 首次运行需要安装依赖
npm run dev
```

**访问地址**：http://localhost:5173

---

## 📖 核心功能使用

### 功能 1：快速分析一只股票

1. 在顶部搜索框输入股票代码（如 `600519`）
2. 点击**"分析"**按钮
3. 等待 1-3 秒，系统自动完成：
   - ✅ 数据采集（多源容灾）
   - ✅ 质量评估
   - ✅ 技术指标计算（MA/MACD/RSI）
   - ✅ 策略执行（内置动量策略）
   - ✅ 方向生成（短线/中线/长线）
   - ✅ 价格结构识别（支撑/压力）
   - ✅ 预测区间计算

4. 查看结果：
   - **页面顶部**：三周期方向卡（偏多/偏空/震荡 + 置信度）
   - **K 线主图**：600px 大图，清晰展示价格走势
   - **支撑/压力 Tab**：关键价位 + 预测区间
   - **流水线 Tab**：查看 10 个阶段的执行详情

### 功能 3：理解三周期方向

#### 短线方向（1-5 交易日）
- **依据**：MACD 柱状线、RSI12、MA5
- **示例**：
  ```
  偏多 · 置信度 72%
  ✓ MACD 柱状线为正值（权重 0.35）
  ✓ 价格在 MA5 上方（权重 0.2）
  ✓ RSI12=45（权重 0.15）
  ⚠ 反证：价格接近压力位
  ```

#### 中线方向（2-12 周）
- **依据**：MA20、30 日涨跌幅、区间位置
- **示例**：
  ```
  震荡 · 置信度 55%
  ✓ 中期趋势向上（MA20 支撑）
  ✓ 30 日涨跌 +3.5%
  ⚠ 反证：价格接近区间上沿，波段或遇阻
  ```

#### 长线方向（3-36 月）
- **依据**：MA60、半年涨跌幅、基本面（后续版本）
- **说明**：目前主要基于技术面，基本面数据将在 Phase 2 补全

### 功能 5：工作流切换

#### 内置工作流
1. **短线攻坚**（默认）
   - 图表周期：3 个月
   - 指标：MA/MACD/RSI
   - 适合：日内/隔日交易

2. **中线波段**
   - 图表周期：1 年
   - 指标：MA/MACD/BOLL
   - 适合：波段持仓 2-12 周

3. **长线价值**
   - 图表周期：3 年
   - 指标：MA/基本面（待补全）
   - 适合：长期投资

#### 切换方法
在顶部控制条，点击**工作流下拉框**，选择目标工作流，系统会：
1. 自动重新运行流水线
2. 使用新工作流的指标配置
3. 更新 K 线图周期
4. 保存选择到 localStorage（刷新后恢复）

### 功能 6：策略输出与修正

#### 内置策略：动量策略
- **逻辑**：
  ```python
  score = MACD柱 × 0.4 + RSI评分 × 0.6
  if RSI < 30: 超卖反弹机会
  if RSI > 70: 超买回调风险
  ```

- **输出示例**：
  ```json
  {
    "score": -0.2,
    "notes": [
      "MACD柱=-0.1411(权重0.4)",
      "RSI超卖(<30.0)"
    ]
  }
  ```

#### 修正策略参数（API 已实现，UI 可扩展）
```bash
curl -X POST http://localhost:8000/api/v1/strategies/builtin_momentum/revise \
  -H "Content-Type: application/json" \
  -d '{
    "params": {"rsi_high": 75, "macd_weight": 0.5, "rsi_low": 25},
    "note": "调整超买超卖阈值"
  }'
```

系统会：
1. 保存新版本到 `strategy_revisions` 表
2. 下次分析自动使用新参数
3. 版本历史可回溯

### 功能 7：上传自定义策略

#### 策略文件格式（strategy.py）
```python
def run(ctx):
    """
    ctx 包含：
    - symbol: str
    - ohlcv: list[dict]  # 最近 60 根 K 线
    - indicators: dict   # {"ma": {...}, "macd": {...}, "rsi": {...}}
    - price_levels: dict # 支撑/压力
    - params: dict       # 用户配置的参数
    """
    close = ctx['ohlcv'][-1]['close']
    ma20 = ctx['indicators']['ma']['ma20']
    
    score = 0.5 if close > ma20 else -0.5
    
    return {
        "score": score,
        "notes": [f"现价 {close} vs MA20 {ma20}"]
    }
```

#### 上传方法（API）
```bash
curl -X POST http://localhost:8000/api/v1/strategies/upload \
  -F "file=@my_strategy.py"
```

## 🎯 高级功能

### 断点续跑
如果流水线在"策略"阶段失败（如策略超时），系统会：
1. 保存前 5 个阶段的结果（采集/质量/市场/质检/特征）
2. 修复策略代码或参数后
3. 点击**"继续上次分析（策略）"**按钮
4. 直接从"策略"阶段继续，节省 1-2 秒数据采集时间

### 市场环境评估
系统自动分析 7 大指数（上证/深证/创业板/科创50/沪深300/中证500/中证1000）：
- **risk**（风险）：多数指数 < MA20，建议降低仓位
- **neutral**（中性）：指数分化，个股为王
- **opportunity**（机会）：多数指数 > MA20，可积极布局

### 数据质量拦截
如果股票数据质量为 D 级（无数据/严重冲突），系统会：
1. 阻断方向生成（显示"数据质量 D 级，不生成方向结论"）
2. 仍保留技术指标、价格结构等基础信息
3. 用户可查看具体问题（"流水线" Tab → "质量"阶段详情）

## 🛠️ 常见问题

### Q1: 后端启动失败？
**检查**：
```bash
# 检查端口占用
lsof -i :8000

# 查看错误日志
cd backend
python -m uvicorn app.main:app --reload
```

**常见原因**：
- Python 版本不兼容（需 Python 3.10+）
- 依赖未安装（`pip install -r requirements.txt`）

---

### Q2: 前端白屏或报错？
**检查**：
```bash
# 确认后端已启动
curl http://localhost:8000/health

# 查看浏览器控制台错误
F12 → Console
```

**常见原因**：
- 后端未启动
- 端口冲突（前端默认 5173，后端默认 8000）
- 依赖版本不匹配（`npm install`）

---

### Q3: 股票数据无法加载？
**检查流水线日志**：
1. 点击**"流水线"** Tab
2. 展开**"阶段详情 / 数据血缘"**
3. 查看"采集"阶段的错误信息

**常见原因**：
- 数据源限流（系统会自动切换到备用源）
- 股票代码错误（如 `60019` 应为 `600019`）
- 缓存目录权限不足

---

### Q4: 方向结论不准确？
**说明**：
- 系统基于技术指标生成方向，不保证准确性
- 市场是复杂系统，技术分析仅提供一种视角
- 建议结合基本面、资金面、政策面综合判断

**改进方法**：
1. 修正策略参数（调整 RSI/MACD 阈值）
2. 上传自定义策略（融入自己的交易逻辑）
3. 查看"反证"提示（如"价格接近压力位"）

---

### Q5: 如何切换数据源优先级？
**编辑配置文件**：
```python
# backend/app/core/config.py
DATA_SOURCES = ["eastmoney", "akshare", "baostock"]  # 默认顺序
```

**重启后端生效**。
