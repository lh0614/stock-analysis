# Task

## Goal
Create a project-specific `RUNBOOK.md` at the repository root for the local stock analysis project.

## Scope
Only create or update `RUNBOOK.md`.

## Out of Scope
Do not modify application code.
Do not modify dependency manifests.
Do not commit git changes.
Do not remove or rewrite existing README files.

## Requirements
The runbook should describe:

- Repository structure.
- Frontend setup and build verification commands.
- Backend setup and smoke-check commands.
- Docker compose startup command.
- Notes about ignored local runtime directories such as `backend/data`, `backend/venv`, `frontend/node_modules`, and `frontend/dist`.
- A minimal manual verification checklist for this project.

## Done When
`RUNBOOK.md` exists and is specific to this project.
